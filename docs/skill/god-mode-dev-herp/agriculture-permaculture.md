# Agriculture & Permaculture

## 1. Fondamentaux Agricoles

### Types d'agriculture
- **Conventionnelle** : intrants chimiques, mécanisation
- **Biologique** : sans pesticides/engrais synthétiques
- **Raisonnée** : réduction intrants, équilibre
- **Permaculture** : conception écosystémique
- **Agroécologie** : principes écologiques appliqués
- **Agroforesterie** : arbres + cultures/élevage

### Sol
- Composition : minéraux, matière organique, air, eau, vie
- Texture : sable, limon, argile (triangle textural)
- Structure : agrégats, porosité
- pH : acide (<7), neutre (7), alcalin (>7)
- Vie du sol : bactéries, champignons, vers, arthropodes

### Amendements
- **Compost** : matière organique décomposée
- **Fumier** : déjections animales + litière
- **BRF** : bois raméal fragmenté
- **Chaux** : correction pH acide
- **Soufre** : correction pH alcalin

## 2. Permaculture

### Éthiques
1. Prendre soin de la Terre
2. Prendre soin des humains
3. Partager équitablement

### Principes de conception (Holmgren)
1. Observer et interagir
2. Capter et stocker l'énergie
3. Obtenir une production
4. Appliquer l'autorégulation
5. Utiliser les ressources renouvelables
6. Ne pas produire de déchets
7. Partir des structures d'ensemble
8. Intégrer plutôt que séparer
9. Utiliser des solutions lentes
10. Utiliser et valoriser la diversité
11. Utiliser les bordures et le marginal
12. Répondre au changement créativement

### Zonage
- **Zone 0** : maison
- **Zone 1** : jardin intensif, herbes, visites fréquentes
- **Zone 2** : verger, petits animaux, compost
- **Zone 3** : grandes cultures, pâturages
- **Zone 4** : semi-sauvage, bois de chauffage
- **Zone 5** : sauvage, observation, réserve

### Secteurs
- Analyse : soleil, vent, eau, accès, vues
- Placement éléments selon secteurs
- Protection, optimisation

## 3. Techniques de Culture

### Buttes
- **Butte sandwich** : lasagne de matériaux
- **Butte Philip Forrer** : bois enterré
- **Hugelkultur** : gros bois en décomposition
- **Butte plate** : culture sur sol préparé

### Paillage (mulch)
- Avantages : humidité, température, adventices, vie sol
- Matériaux : paille, foin, BRF, feuilles, carton
- Épaisseur : 10-20 cm selon matériau

### Associations de plantes
| Plante | Bonnes associations | Mauvaises |
|--------|---------------------|-----------|
| Tomate | Basilic, carotte, persil | Fenouil, chou |
| Carotte | Poireau, oignon, laitue | Aneth |
| Haricot | Maïs, courge, pomme de terre | Oignon, ail |
| Chou | Haricot, céleri, camomille | Fraise, tomate |
| Courge | Maïs, haricot | Pomme de terre |

### Rotation des cultures
- Objectif : éviter épuisement, maladies
- Familles : légumes-feuilles → légumineuses → légumes-fruits → légumes-racines
- Durée : 4 ans minimum avant retour même famille

### Engrais verts
- Légumineuses : trèfle, vesce, féverole (azote)
- Crucifères : moutarde, radis fourrager (structuration)
- Graminées : seigle, avoine (biomasse)
- Phacélie : mellifère, tout sol

| Poirier | Soleil | Croisée | 4-6 ans |

### Petits fruits
- **Framboisier** : remontant/non-remontant
- **Groseillier, cassissier** : ombre partielle OK
- **Myrtillier** : sol acide obligatoire
- **Fraisier** : variétés remontantes/non-remontantes

### Taille
- **Formation** : structure jeune arbre
- **Fructification** : stimuler production
- **Entretien** : aération, suppression bois mort
- Époques : hiver (dormance), été (vert)

### Forêt comestible
- Strates : canopée, sous-canopée, arbustes, herbacées, couvre-sol, grimpantes, racines
- Guildes : associations bénéfiques autour d'un arbre

## 6. Élevage Petit

### Poules
- Besoins : 10-20 m²/poule, poulailler isolé
- Alimentation : grains, verdure, déchets cuisine, calcium
- Production : 150-300 œufs/an selon race
- Races : Marans, Sussex, Leghorn, Poule soie

### Canards
- Plus rustiques, moins de maladies
- Besoins en eau (bassin)
- Excellents contre limaces
- Races : Coureur indien (ponte), Pékin (chair)

### Lapins
- Alimentation : foin, herbe, légumes
- Reproduction rapide
- Fumier excellent pour jardin
- Clapier ou garenne

### Abeilles
- Ruches : Dadant, Warré, Langstroth, Kenyane
- Production : miel, cire, pollinisation
- Réglementation : déclaration, distances
- Formation recommandée

## 7. Gestion de l'Eau

### Récupération eau de pluie
- Calcul : surface toiture × pluviométrie × 0.9
- Stockage : cuves, citernes, mares
- Filtration : crapaudine, filtre

### Irrigation
- **Goutte-à-goutte** : économe, localisé
- **Aspersion** : couverture large
- **Oyas** : poteries enterrées, diffusion lente
- **Mulch** : réduction évaporation 70%

### Mares et bassins
- Biodiversité, réserve eau, microclimat
- Étanchéité : argile, bâche EPDM
- Plantes aquatiques, faune

### Swales (baissières)
- Fossés de rétention sur courbes de niveau
- Infiltration eau dans sol
- Plantations sur berme aval

## 8. Compostage

### Compost classique
- Ratio C/N : 30:1 idéal
- Carbone (bruns) : feuilles, paille, carton
- Azote (verts) : tontes, épluchures, fumier
- Aération, humidité
- Durée : 6-12 mois

### Vermicompostage
- Vers : Eisenia fetida (vers du fumier)
- Intérieur possible
- Lombricompost très riche
- Thé de compost (engrais liquide)

### Bokashi
- Fermentation anaérobie
- EM (micro-organismes efficaces)
- Rapide, peu d'odeurs
- Tous déchets cuisine (même viande)

### Compost de surface
- Dépôt direct sur sol
- Décomposition lente
- Vie du sol active

## 9. Protection des Cultures

### Lutte biologique
- Auxiliaires : coccinelles, chrysopes, syrphes
- Nématodes : contre limaces, vers
- Bacillus thuringiensis : chenilles
- Purins : ortie, prêle, consoude

### Préparations naturelles
| Préparation | Usage |
|-------------|-------|
| Purin d'ortie | Fertilisant, insectifuge |
| Décoction prêle | Fongicide |
| Purin consoude | Fertilisant (K) |
| Macération ail | Insectifuge, fongicide |
| Savon noir | Pucerons |

### Barrières physiques
- Voiles anti-insectes
- Filets oiseaux
- Cloches, tunnels
- Pièges (limaces, carpocapse)

### Plantes compagnes protectrices
- Œillets d'Inde : nématodes
- Basilic : mouches, moustiques
- Capucines : pucerons (plante piège)
- Tanaisie : insectes divers

## 10. Autonomie Alimentaire

### Surface nécessaire
- 200-400 m² par personne (légumes)
- Variables : climat, techniques, temps disponible

### Planification
- Besoins famille : quantités par légume
- Étalement récoltes
- Conservation : cave, conserves, congélation

### Conservation
- **Cave** : pommes de terre, courges, carottes (sable)
- **Séchage** : tomates, haricots, herbes
- **Lactofermentation** : choucroute, kimchi
- **Conserves** : stérilisation
- **Congélation** : blanchiment préalable

### Semences
- Variétés anciennes, reproductibles
- Récolte, séchage, stockage
- Réseau échanges
