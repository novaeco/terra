# Aquariophilie

## 1. Types d'Aquariums

### Eau douce tropicale
- Température : 24-28°C
- pH : variable selon biotope
- Le plus courant, grande diversité

### Eau douce froide
- Température : 15-22°C
- Poissons rouges, carpes, espèces tempérées
- Pas de chauffage nécessaire

### Eau de mer (récifal)
- Salinité : 35 g/L
- Coraux, poissons marins, invertébrés
- Techniquement exigeant, coûteux

### Eau saumâtre
- Salinité intermédiaire
- Archers, monos, scats, certains gobies

### Biotopes spécifiques
- Amazonien : acide, tannins, bois
- Africain (cichlidés) : alcalin, rocheux
- Asiatique : planté, calme

## 2. Paramètres de l'Eau

### Paramètres physiques
| Paramètre | Eau douce | Marin |
|-----------|-----------|-------|
| Température | 24-28°C | 24-26°C |
| pH | 6.0-8.0 | 8.0-8.4 |
| dGH (dureté totale) | 4-20° | - |
| dKH (dureté carbonatée) | 4-12° | 7-12° |

### Paramètres chimiques
| Paramètre | Idéal | Toxique |
|-----------|-------|---------|
| Ammoniac (NH₃) | 0 | >0.02 mg/L |
| Nitrites (NO₂⁻) | 0 | >0.5 mg/L |
| Nitrates (NO₃⁻) | <25 mg/L | >50 mg/L |

### Cycle de l'azote
```
Déchets → NH₃ (ammoniac) → NO₂⁻ (nitrites) → NO₃⁻ (nitrates)
         (Nitrosomonas)    (Nitrobacter)      (changements d'eau/plantes)
```
- Cyclage : 4-8 semaines avant introduction poissons
- Pics toxiques si cycle non établi

## 3. Équipement

### Filtration
- **Mécanique** : mousse, ouate (particules)
- **Biologique** : céramique, bio-balls (bactéries)
- **Chimique** : charbon actif, résines

Types de filtres :
- Interne : petit volume, discret
- Externe (canister) : grand volume, efficace
- Décantation : aquarium marin
- Exhausteur : air, petit bac
- HMF (hamburger mattenfilter) : mousse

### Chauffage
- Combiné thermo-chauffant
- 1W/L (eau douce), plus pour marin
- Thermostat, sonde séparée

### Éclairage
- LED : économique, modulable, durée de vie
- T5/T8 : fluorescent, bons pour plantes
- HQI/HID : récifal, puissant
- Photopériode : 8-12h/jour

### Autres équipements
- Pompe de brassage
- Écumeur (marin)
- CO₂ (aquarium planté)
- Osmoseur (eau pure)

## 4. Poissons d'Eau Douce

### Communautaires faciles
| Espèce | Taille | Température | Groupe |
|--------|--------|-------------|--------|
| Guppy | 4-6 cm | 22-28°C | 1M:2-3F |
| Néon | 3-4 cm | 20-26°C | 10+ |
| Platy | 5-6 cm | 20-28°C | 1M:2-3F |
| Corydoras | 5-7 cm | 22-26°C | 6+ |
| Ancistrus | 10-15 cm | 22-28°C | 1-2 |
| Rasbora | 3-5 cm | 22-26°C | 10+ |

### Cichlidés africains (lac Malawi/Tanganyika)
- Mbuna, haplos, peacocks
- pH : 7.5-8.5, dureté élevée
- Territoriaux, surpopulation recommandée

### Cichlidés américains
- Scalaires, discus, ramirezi
- pH : 5.5-7.0, eau douce à très douce
- Discus : exigeant, température élevée (28-30°C)

### Labyrinthidés
- Bettas, gouramis
- Organe labyrinthe (air atmosphérique)
- Betta : solitaire, territorial

### Cyprinidés
- Barbus, danios, rasboras
- Actifs, bancs

## 5. Poissons Marins

### Faciles
- Amphiprions (poissons-clowns)
- Chrysiptera (demoiselles)
- Gobiidés
- Pseudochromis

### Intermédiaires
- Acanthuridés (chirurgiens)
- Pomacanthidés (poissons-anges)
- Labridés (labres)

### Difficiles
- Chaetodontidés (papillons) - corallivores
- Syngnathidés (hippocampes)
- Requins, raies

### Invertébrés marins
- Coraux mous (débutants), coraux durs (SPS, LPS)
- Crevettes : Lysmata, Stenopus
- Anémones (attention compatibilité)
- Étoiles, oursins, ophiures

## 6. Aquarium Planté

### Plantes faciles
- Anubias, Cryptocoryne, Microsorum
- Mousse de Java, Vesicularia
- Vallisneria, Sagittaria
- Pas de CO₂ obligatoire

### Plantes exigeantes
- Hemianthus, Glossostigma (tapis)
- Rotala, Ludwigia (couleurs)
- CO₂, éclairage fort, fertilisation

### Paramètres plantés
- Éclairage : 30-50 lumens/L (moyen), 50+ (intense)
- CO₂ : 20-30 mg/L (injection)
- Fertilisation : NPK + oligoéléments
- Substrat nutritif : aquasoil, substrat technique

### Hardscape
- Roches : pas calcaires pour eau douce acide
- Racines : spider, mopani (tannins)
- Aquascaping : styles Nature, Iwagumi, Dutch

## 7. Entretien

### Hebdomadaire
- Changement d'eau : 20-30%
- Nettoyage vitres
- Taille plantes si nécessaire
- Tests eau (débutants)

### Mensuel
- Nettoyage filtre (eau de l'aquarium !)
- Aspiration sol (attention plantes)
- Vérification équipement

### Alimentation
- 1-2 fois/jour, petites quantités
- Variété : flocons, granulés, congelé, vivant
- Jour de jeûne hebdomadaire

## 8. Maladies & Traitements

### Maladie des points blancs (Ich)
- Ichthyophthirius multifiliis
- Points blancs, frottements
- Traitement : température + sel/médicament

### Pourriture des nageoires
- Bactérien (Aeromonas, Pseudomonas)
- Nageoires effilochées
- Traitement : eau propre, antibactérien

### Hydropisie
- Ventre gonflé, écailles hérissées
- Insuffisance rénale, infection
- Souvent fatal

### Oodinium (velours)
- Parasite, aspect velours doré
- Traitement : cuivre, obscurité

### Prévention
- Quarantaine nouveaux arrivants
- Paramètres stables
- Pas de surpopulation
- Alimentation variée

## 9. Aquariophilie Marin Avancée

### Système récifal
- Éclairage : LED haute puissance, spectre coraux
- Écumeur : élimination matières organiques
- Réacteur à calcium/kalkwasser
- Balling : supplémentation Ca, Mg, KH

### Paramètres récifal
| Paramètre | Valeur |
|-----------|--------|
| Salinité | 35 ppt |
| Calcium | 400-450 ppm |
| Magnésium | 1300-1400 ppm |
| KH | 7-11 dKH |
| Phosphates | <0.03 ppm |
| Nitrates | <5 ppm |

### Coraux
- Mous : Sarcophyton, Sinularia, Xenia (faciles)
- LPS : Euphyllia, Acanthastrea, Fungia
- SPS : Acropora, Montipora, Pocillopora (exigeants)
