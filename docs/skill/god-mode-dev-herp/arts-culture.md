# Arts, Culture & Langues

## 1. Histoire de l'Art

### Préhistoire & Antiquité
- Art pariétal : Lascaux, Altamira
- Mésopotamie : ziggurats, cunéiforme
- Égypte : pyramides, hiéroglyphes, style frontal
- Grèce : ordres (dorique, ionique, corinthien), sculpture idéalisée
- Rome : architecture (arcs, voûtes, dômes), mosaïques

### Moyen Âge
- Art byzantin : icônes, Sainte-Sophie
- Art roman : églises massives, tympans sculptés
- Art gothique : arcs brisés, vitraux, cathédrales
- Enluminure, orfèvrerie

### Renaissance (XV-XVI)
- Perspective, anatomie, humanisme
- Quattrocento : Botticelli, Fra Angelico
- Cinquecento : Léonard, Michel-Ange, Raphaël
- Renaissance nordique : Van Eyck, Dürer

### Baroque & Classicisme (XVII)
- Baroque : mouvement, émotion, clair-obscur (Caravage, Rubens, Bernin)
- Classicisme français : Poussin, Le Brun, Versailles

### XVIII-XIX siècles
- Rococo : légèreté, ornements (Watteau, Fragonard)
- Néoclassicisme : retour antique (David)
- Romantisme : émotion, nature (Delacroix, Turner)
- Réalisme : quotidien (Courbet)
- Impressionnisme : lumière, instant (Monet, Renoir, Degas)
- Post-impressionnisme : Van Gogh, Cézanne, Gauguin

### Art moderne (XX)
- Fauvisme : couleurs vives (Matisse)
- Expressionnisme : émotion intense (Munch, Kirchner)
- Cubisme : géométrie, perspectives multiples (Picasso, Braque)
- Futurisme : mouvement, modernité
- Dadaïsme : anti-art (Duchamp)
- Surréalisme : inconscient, rêves (Dalí, Magritte)
- Abstraction : Kandinsky, Mondrian, Rothko
- Pop Art : culture populaire (Warhol, Lichtenstein)

### Art contemporain
- Minimalisme, art conceptuel
- Performances, installations
- Art vidéo, numérique
- Street art

## 2. Musique

### Éléments fondamentaux
- Mélodie, harmonie, rythme
- Timbre, dynamique
- Tonalité, modes, gammes
- Notation, solfège

### Histoire de la musique occidentale
- Antiquité : modes grecs
- Moyen Âge : plain-chant, polyphonie
- Renaissance : musique vocale, madrigaux
- Baroque : Bach, Vivaldi, Haendel (fugue, concerto)
- Classique : Haydn, Mozart, Beethoven (sonate, symphonie)
- Romantique : Chopin, Brahms, Wagner (virtuosité, émotion)
- XX siècle : Debussy, Stravinsky, Schoenberg (atonalité, dodécaphonisme)
- Contemporain : minimalisme, électronique, spectral

### Musiques populaires
- Blues, jazz : origines afro-américaines
- Rock'n'roll, rock, punk, metal
- Pop, R&B, soul, hip-hop
- Électronique : house, techno, EDM
- Musiques du monde

### Instruments
- Cordes : frottées (violon), pincées (guitare), frappées (piano)
- Vents : bois (flûte, clarinette), cuivres (trompette)
- Percussions : membranophones, idiophones
- Électroniques : synthétiseurs, samplers
