# Bricolage & DIY

## 1. Menuiserie

### Bois - Types et usages
| Type | Caractéristiques | Usages |
|------|------------------|--------|
| Pin | Tendre, économique | Charpente, meubles simples |
| Sapin | Léger, peu durable ext. | Intérieur, coffrage |
| Chêne | Dur, durable, tanins | Meubles, parquet, ext. |
| Hêtre | Dur, clair, se travaille bien | Meubles, outils |
| Noyer | Noble, foncé | Ébénisterie |
| MDF | Panneau fibres | Meubles, peinture |
| Contreplaqué | Plis croisés, stable | Structure, plans de travail |
| OSB | Copeaux orientés | Structure, sous-couche |

### Assemblages
- **Vissage** : rapide, démontable
- **Clouage** : rapide, définitif
- **Collage** : invisible, solide (colle bois PVA, polyuréthane)
- **Tourillons** : alignement, renfort
- **Lamelles (domino, biscuit)** : alignement, renfort
- **Tenon-mortaise** : traditionnel, très solide
- **Queue d'aronde** : tiroirs, ébénisterie
- **Mi-bois** : croisement, cadres

### Outils essentiels
- Mesure : mètre, équerre, niveau
- Coupe : scie égoïne, scie circulaire, scie sauteuse
- Perçage : perceuse-visseuse, forets bois
- Finition : ponceuse, râpe, ciseaux à bois
- Serrage : serre-joints, presses

### Finitions bois
- **Ponçage** : grains croissants (80→120→180→240)
- **Vernis** : protection, brillant
- **Huile** : naturel, entretien régulier
- **Cire** : aspect traditionnel
- **Lasure** : extérieur, protection UV
- **Peinture** : opaque, primaire nécessaire

- Disjoncteur, différentiel 30mA

## 6. Carrelage

### Types
- Grès cérame : résistant, tout usage
- Faïence : murs, moins résistant
- Pierre naturelle : esthétique, entretien
- Mosaïque : déco, technique

### Pose
1. Préparation support (plan, propre)
2. Calepinage (plan de pose)
3. Encollage (simple ou double)
4. Pose, croisillons
5. Joints (24-48h après)
6. Nettoyage

### Colle et joints
- Colle C1, C2 (standard, améliorée)
- Colle flex : grands carreaux, plancher chauffant
- Joints : ciment, époxy
- Largeur joints : selon carreau et usage

### Découpe
- Carrelette manuelle : droites
- Meuleuse + disque diamant : coupes complexes
- Scie cloche : passages tuyaux

## 7. Isolation & Placo

### Isolants
| Matériau | λ (W/m.K) | Usages |
|----------|-----------|--------|
| Laine de verre | 0.032-0.040 | Combles, murs |
| Laine de roche | 0.034-0.040 | Tout, feu |
| Polystyrène exp. | 0.030-0.038 | Sols, ITE |
| Polyuréthane | 0.022-0.028 | Haute perf., toiture |
| Ouate cellulose | 0.038-0.042 | Soufflage combles |
| Fibre bois | 0.038-0.043 | Écologique |

### Résistance thermique
- R = épaisseur / λ
- RT2012 : R ≥ 4 (murs), R ≥ 8 (toiture)
- RE2020 : exigences renforcées

### Placo (plâtre)
- BA13 : standard 12.5mm
- Hydrofuge (vert) : pièces humides
- Phonique (bleu) : acoustique
- Feu (rose) : résistance feu

### Pose placo
1. Ossature : rails, montants
2. Isolation entre montants
3. Vissage plaques (vis placo)
4. Bandes et enduit joints
5. Ponçage, finition
