# Biologie des Reptiles

## 1. Anatomie Générale

### Tégument
- **Écailles** : kératinisées, épidermiques
  - Types : lisses, carénées, granulaires, tuberculeuses
  - Arrangement : imbriquées, juxtaposées
- **Mue (ecdysis)** : renouvellement périodique complet
  - Serpents : en une pièce (exuvie)
  - Lézards : en lambeaux
  - Fréquence : croissance, saison, nutrition
- **Coloration** : chromatophores (mélanophores, xanthophores, iridophores)

### Squelette
- **Crâne** : diapside (2 fosses temporales), kinétisme chez serpents
- **Colonne vertébrale** : nombreuses vertèbres, côtes mobiles
- **Membres** : pentadactyles (réduits/absents chez serpents, certains lézards)
- **Carapace tortues** : fusion vertèbres + côtes + ostéodermes

### Système musculaire
- Musculature axiale développée (locomotion ondulante)
- Muscles costaux pour respiration
- Muscles constricteurs puissants (Boidae, Pythonidae)

## 2. Systèmes Physiologiques

### Système respiratoire
- Poumons : unilobé (serpents : droit développé, gauche vestigial) ou bilobés
- Pas de diaphragme : pompe costale
- Trachée avec anneaux cartilagineux
- **Glotte** : fermeture lors déglutition proies

### Système circulatoire
- Cœur : 3 chambres (2 oreillettes, 1 ventricule partiellement cloisonné)
- Crocodiliens : 4 chambres vraies + foramen de Panizza
- Système porte rénal : sang membres postérieurs → reins
- Implications : sites d'injection (éviter membres postérieurs)

### Système digestif
- **Serpents** : mâchoires mobiles, ligament quadrato-mandibulaire élastique
- Dents : acrodontes (agames), pleurodontes (iguanes, serpents), thécodontes (croco)
- Estomac : acide puissant (digestion os, kératine)
- Caecum : développé chez herbivores (iguanes)
- Cloaque : chambre commune (digestif, urinaire, reproducteur)

### Système excréteur
- Reins métanéphros
- Excrétion : acide urique (semi-solide, économie eau)
- Vessie : présente tortues/lézards, absente serpents
- Glandes à sel : iguanes marins, certaines tortues

### Thermorégulation (ectothermie)
- **POTZ** (Preferred Optimal Temperature Zone) : spécifique à l'espèce
- Comportements : basking, shuttling, thigmothermie
- Gradient thermique nécessaire en captivité
- **Fever behavior** : recherche chaleur lors infection

### Système nerveux & sens
- **Vision** : variable selon espèce
  - Fossettes thermosensibles (Boidae, Crotalinae) : infrarouge
  - Vision UV (certains lézards)
- **Olfaction** : organe de Jacobson (voméronasal)
  - Langue bifide : collecte molécules
- **Audition** : pas d'oreille externe, sensibilité vibrations sol
- **Lignes latérales** : crocodiliens (détection mouvements eau)

## 3. Reproduction

### Modes reproducteurs
- **Ovipare** : ponte d'œufs (majorité)
- **Vivipare** : naissance jeunes développés (Boa constrictor)
- **Ovovivipare** : œufs incubés intérieurement (terme désuet)

### Détermination du sexe
- **TSD** (Temperature-dependent Sex Determination)
  - Tortues : généralement femelles à T° élevées
  - Crocodiliens : mâles à T° intermédiaires
  - Lézards : variable selon espèces
- **GSD** (Genetic Sex Determination)
  - Serpents : chromosomes ZW (femelles hétérogamétiques)
  - Certains lézards : XY

### Dimorphisme sexuel
- **Serpents** : mâles queue plus longue (hémipénis), parfois plus petits
- **Lézards** : mâles souvent plus grands, colorés, crêtes, pores fémoraux
- **Tortues** : plastron concave mâles, queue plus longue
- **Crocodiliens** : mâles significativement plus grands

### Cycle reproducteur
- **Période de repos** (brumation/hivernation simulée)
- **Conditionnement** : variations photopériode, température, alimentation
- **Accouplement** : combats rituels mâles, phéromones
- **Hémipénis** : organe copulateur pair, un seul utilisé
- **Spermathèque** : stockage sperme (années possible)

### Ponte & incubation
- **Substrat ponte** : vermiculite, perlite humide
- **Paramètres** : température, humidité critiques
- **Durée** : variable (45-90 jours typique serpents)
- **Éclosion** : dent de l'œuf (caroncule)

## 4. Comportement

### Comportements thermorégulateurs
- Basking (exposition soleil/chaleur)
- Shuttling (alternance zones chaudes/froides)
- Postures : aplatissement, élévation
- Gaping (bouche ouverte dissipation chaleur)

### Comportements alimentaires
- **Embuscade** : pythons, vipères
- **Recherche active** : couleuvres, varans
- **Constriction** : Boidae, Pythonidae, certains Colubridae
- **Venin** : immobilisation, prédigestion
- **Déglutition** : toujours tête première (écailles proie)

### Comportements défensifs
- Fuite, immobilité (camouflage)
- Postures menaçantes : gonflement, sifflement
- Musking : sécrétion cloacale nauséabonde
- Thanatose : simulation mort (Heterodon)
- Autotomie : perte queue (lézards)
- Saignement oculaire : Phrynosoma

### Comportements sociaux
- Majorité solitaires
- Agrégations hivernales (serpents)
- Territorialité (lézards mâles)
- Soins parentaux : crocodiliens, certains scinques

## 5. Écologie & Habitat

### Types d'habitats
- **Terrestre** : sol, terriers
- **Arboricole** : canopée, branches
- **Fossorial** : souterrain
- **Aquatique/semi-aquatique** : eau douce, marine
- **Saxicole** : rocheux
- **Désertique, tropical, tempéré**

### Adaptations
- **Arboricoles** : queue préhensile, doigts adhésifs, corps comprimé latéralement
- **Fouisseurs** : yeux réduits, corps cylindrique, museau adapté
- **Aquatiques** : valves nasales, queue comprimée, palmures
- **Désertiques** : écailles granuleuses, locomotion sidewinding

### Régimes alimentaires
- **Carnivores stricts** : serpents
- **Insectivores** : majorité lézards
- **Herbivores** : Iguana, Uromastyx, tortues terrestres adultes
- **Omnivores** : Pogona, Tiliqua, certaines tortues
- **Spécialistes** : œufs (Dasypeltis), escargots, crabes
