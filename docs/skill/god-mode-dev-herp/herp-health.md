# Santé & Pathologies des Reptiles

## 1. Signes de Bonne Santé

### Aspect général
- Yeux clairs, brillants (sauf pré-mue)
- Écailles lisses, sans blessures ni rétentions de mue
- Corps bien charnu (vertèbres non visibles)
- Respiration calme, bouche fermée
- Comportement actif et réactif

### Indicateurs positifs
- Alimentation régulière
- Selles formées : partie blanche (urates) + brune (fèces)
- Mue complète en une pièce (serpents)
- Posture normale, mobilité fluide

## 2. Urgences Vétérinaires

### Urgences absolues
- Prolapsus cloacal
- Détresse respiratoire sévère
- Dystocie (rétention œufs > 24-48h)
- Traumatisme grave, fractures ouvertes
- Brûlures étendues
- Convulsions, paralysie soudaine
- Hémorragie importante

### Premiers soins
- **Prolapsus** : maintenir humide (sérum phy), ne pas repousser
- **Brûlures** : rincer eau tiède, pas de crèmes
- **Hypothermie** : réchauffement progressif
- **Hyperthermie** : refroidissement progressif

## 3. Pathologies Respiratoires

### Infection respiratoire (RI)
**Causes** : bactéries, virus, température inadéquate, stress
**Symptômes** : respiration bouche ouverte, wheezing, bulles nasales, mucus, stargazing
**Traitement** : antibiotiques (Baytril, Fortaz), nébulisation, correction paramètres

### Stomatite (mouth rot)
**Symptômes** : gencives enflées, pus, salivation, anorexie
**Traitement** : débridement, Betadine diluée, antibiotiques systémiques

## 4. Parasites

### Internes
- **Cryptosporidium** : fatal serpents, régurgitations, pas de traitement efficace
- **Amibes** : diarrhée mucoïde → métronidazole
- **Coccidies** : diarrhée sanglante → sulfamides
- **Nématodes** : → fenbendazole
- **Cestodes** : → praziquantel

### Externes
- **Acariens (Ophionyssus)** : points noirs mobiles, anémie
  - Traitement animal + environnement complet, 3 semaines
- **Tiques** : retrait pince, rotation, désinfection

## 5. Pathologies Métaboliques

### MBD (Metabolic Bone Disease)
**Causes** : carence Ca, D3, manque UVB
**Symptômes** : os mous, déformations, mâchoire caoutchouteuse, tremblements, fractures
**Prévention** : UVB adapté, calcium, ratio Ca:P 2:1
**Traitement** : calcium injectable, correction régime/UVB

### Goutte
**Causes** : déshydratation, excès protéines, insuffisance rénale
**Symptômes** : gonflements articulaires
**Traitement** : allopurinol, hydratation

### Hypovitaminose A
**Espèces** : tortues aquatiques
**Symptômes** : paupières gonflées
**Traitement** : vitamine A injectable

## 6. Pathologies Digestives

### Anorexie
**Physiologique** : mue, reproduction, brumation, gravité
**Pathologique** : stress, parasites, infection, occlusion

### Régurgitation
**Causes** : manipulation post-repas, T° basse, stress, Crypto
**Gestion** : attendre 10-14j, proie plus petite, vérifier T°

### Occlusion
**Causes** : substrat ingéré, grosse proie, déshydratation
**Symptômes** : absence selles, gonflement, anorexie
**Traitement** : bains tièdes, vétérinaire si persiste

### Prolapsus
**Types** : intestin, oviducte, hémipénis
**Urgence** : maintenir humide, vétérinaire immédiat

## 7. Pathologies Reproductives

### Dystocie
**Facteurs** : premier ponte, malnutrition, MBD, absence site ponte
**Symptômes** : efforts > 24-48h, léthargie, œuf visible
**Traitement** : bain tiède, oxytocine, chirurgie

## 8. Pathologies Tégumentaires

### Rétention de mue
**Causes** : humidité insuffisante, parasites
**Zones critiques** : yeux (lunettes), doigts
**Traitement** : bains, chambre humide, retrait délicat

### Brûlures thermiques
**Causes** : contact sources chaleur non protégées
**Prévention** : grilles, thermostats

### Scale rot / Blister disease
**Causes** : substrat humide/sale
**Traitement** : correction environnement, antiseptiques

### Abcès
**Caractéristiques** : pus caséeux solide chez reptiles
**Traitement** : excision chirurgicale (pas drainage)

## 9. Problèmes Neurologiques

### IBD (Inclusion Body Disease)
**Affectés** : Boidae (boas, pythons)
**Agent** : arenavirus
**Symptômes** : stargazing, désorientation, régurgitations, paralysie
**Issue** : fatal, euthanasie recommandée
**Prévention** : quarantaine stricte

### Wobble syndrome
**Affectés** : Python regius morph Spider
**Symptômes** : tremblements tête, désorientation
**Nature** : génétique, lié gène Spider

## 10. Soins Vétérinaires

### Examens diagnostiques
- Coproscopie : parasites
- Radiographie : squelette, œufs
- Échographie : organes
- PCR, cultures : infections

### Médicaments courants
| Médicament | Usage |
|------------|-------|
| Baytril (enrofloxacine) | Antibiotique large spectre |
| Fortaz (ceftazidime) | Antibiotique Gram- |
| Panacur (fenbendazole) | Nématodes |
| Flagyl (métronidazole) | Protozoaires |
| Droncit (praziquantel) | Cestodes |

### Administration
- Orale : sonde gastrique, dans proie
- Injectable : IM membre antérieur (éviter postérieur : système porte rénal)
- Nébulisation : infections respiratoires
