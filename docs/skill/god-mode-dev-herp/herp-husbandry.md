# Élevage & Terrariophilie

## 1. Types d'Installations

### Terrariums
- **Verre** : visibilité, inertie thermique, poids
- **PVC/mélaminé** : léger, isolant, empilable (rack systems)
- **Grillage** : ventilation maximale (caméléons)
- **Bois** : isolation, mais sensible humidité

### Dimensions recommandées
- Règle générale : longueur ≥ longueur totale animal
- **Serpents terrestres** : longueur × 0.5 × 0.5 (L×l×h)
- **Serpents arboricoles** : hauteur prioritaire
- **Lézards terrestres** : surface sol prioritaire
- **Lézards arboricoles** : hauteur prioritaire
- **Tortues terrestres** : 8-10× longueur carapace (enclos)
- **Tortues aquatiques** : volume eau 10L/2.5cm carapace minimum

## 2. Paramètres Environnementaux

### Température
- **Gradient thermique** obligatoire : zone chaude + zone froide
- **Point chaud (basking)** : selon espèce
- **Zone froide** : permet thermorégulation
- **Baisse nocturne** : naturelle, souvent bénéfique

| Groupe | Point chaud | Zone froide | Nuit |
|--------|-------------|-------------|------|
| Python regius | 32-34°C | 26-28°C | 24-26°C |
| Boa constrictor | 32-34°C | 26-28°C | 24-26°C |
| Pantherophis guttatus | 30-32°C | 24-26°C | 20-24°C |
| Pogona vitticeps | 40-45°C | 28-30°C | 20-24°C |
| Eublepharis macularius | 32-34°C | 24-26°C | 22-24°C |
| Testudo hermanni | 30-35°C | 20-24°C | 15-20°C |

### Sources de chaleur
- **Lampes céramique** : chaleur sans lumière (nuit)
- **Lampes halogènes/incandescentes** : chaleur + lumière
- **Tapis chauffants** : thermoconduction, sous terrarium
- **Câbles chauffants** : flexibilité
- **Lampes deep heat projector** : chaleur infrarouge pénétrante
- **Thermostat** : obligatoire, on/off ou proportionnel

### Éclairage
- **Photopériode** : 12h/12h standard, variations saisonnières
- **UVB** : essentiel espèces diurnes, synthèse vitamine D3
  - Indice Ferguson : zones 1-4 selon espèce
  - Zone 1 : ombrage (serpents, geckos nocturnes)
  - Zone 2-3 : exposition modérée (Pogona, Uromastyx)
  - Zone 4 : forte exposition (Uromastyx, certaines tortues)
- **UVA** : comportement, vision, appétit
- **Tubes T5/T8, lampes Mercury Vapor (MVB)** : sources UVB
- **Distance tube** : selon % UVB (10.0 plus loin que 5.0)
- **Remplacement** : tous 6-12 mois (dégradation UVB)

### Humidité
- **Hygrométrie** : variable selon espèce et habitat naturel
  - Désertique : 30-40%
  - Tropical : 60-80%
  - Mue : augmentation temporaire bénéfique
- **Contrôle** : pulvérisation, bac d'eau, substrat humide, ventilation
- **Mesure** : hygromètre fiable, calibré

| Groupe | Humidité |
|--------|----------|
| Pogona vitticeps | 30-40% |
| Eublepharis macularius | 40-50% (boîte humide) |
| Python regius | 50-60% (70-80% mue) |
| Morelia viridis | 60-80% |
| Correlophus ciliatus | 60-80% |

## 3. Aménagement

### Substrats
- **Papier journal/sopalin** : quarantaine, hygiène, économique
- **Copeaux de hêtre** : absorbant, serpents
- **Coco (fibre, chips, humus)** : tropical, retient humidité
- **Terre de bruyère** : tortues, naturel
- **Sable** : désertique (risque ingestion)
- **Excavator clay** : fouisseurs
- **À éviter** : copeaux cèdre/pin (toxiques), substrats poussiéreux

### Cachettes
- **Minimum 2** : zone chaude et zone froide
- Taille adaptée : animal doit toucher les parois
- Matériaux : liège, coco, résine, carton (jetable)

### Enrichissement
- **Branches** : arboricoles (sécurisées, désinfectées)
- **Plantes** : vraies ou artificielles (non toxiques)
- **Pierres** : basking, usure griffes
- **Structures grimper** : lianes, plateformes

### Points d'eau
- **Bac** : taille adaptée (bain complet si nécessaire)
- **Eau propre** : changement fréquent
- **Profondeur** : selon espèce (tortues : nage, serpents : bain)

## 4. Alimentation

### Serpents
- **Proies adaptées** : largeur proie ≈ partie la plus large du serpent
- **Fréquence** : variable selon âge, espèce, taille proie
  - Juvéniles : tous les 5-7 jours
  - Adultes : tous les 10-21 jours
- **Proies congelées/décongelées** : recommandé (sécurité)
- **Proies vivantes** : risques blessures, stress
- **Jeûne** : normal en période de mue, reproduction, brumation

### Lézards insectivores
- **Variété** : grillons, criquets, blattes, vers de farine, vers à soie
- **Gut-loading** : alimentation proies 24-48h avant
- **Calcium** : saupoudrage chaque repas (juvéniles), 2-3× semaine (adultes)
- **Vitamines** : 1× semaine (D3 si pas UVB)
- **Taille proies** : < espace inter-oculaire

### Lézards herbivores/omnivores
- **Verdure** : endives, mâche, pissenlit, roquette, feuilles de mûrier
- **Légumes** : courges, carottes râpées, haricots verts
- **Fruits** : occasionnels (10-15%), riches en sucres
- **Protéines** : insectes (Pogona jeunes), éviter adultes Uromastyx
- **À éviter** : épinards, laitue iceberg, avocat, rhubarbe

### Tortues terrestres
- **Herbivores stricts** : herbes, fleurs, feuilles
- **Calcium** : os de seiche, saupoudrage
- **Fibres** : essentielles (foin, herbes sèches)
- **À éviter** : fruits (sauf rare), protéines animales

### Tortues aquatiques
- **Carnivores (jeunes)** : insectes aquatiques, poissons, crevettes
- **Omnivores (adultes)** : végétaux augmentent avec l'âge
- **Granulés** : complément, pas base exclusive

### Crocodiliens
- **Carnivores stricts** : rongeurs, volaille, poissons
- **Alimentation complète** : proies entières avec os
- **Fréquence** : 1-2× semaine adultes

## 5. Maintenance

### Hygiène quotidienne
- Retirer déjections, mues, restes nourriture
- Vérifier eau (propreté, niveau)
- Observer animal (comportement, aspect)

### Hygiène hebdomadaire
- Nettoyer vitres
- Vérifier matériel (thermostats, lampes)
- Changer substrat zones souillées

### Hygiène mensuelle/périodique
- Changement substrat complet
- Désinfection décors, cachettes
- Vérification paramètres (thermomètres, hygromètres)

### Quarantaine
- **Durée** : 60-90 jours minimum nouveaux arrivants
- **Installation** : simple, facile à désinfecter
- **Observation** : comportement, alimentation, selles, parasites

## 6. Reproduction en Captivité

### Conditionnement
- **Sexage** : sondage (serpents), éversion (lézards), palpation, dimorphisme
- **Période repos** : brumation simulée (baisse T°, photopériode, jeûne)
- **Durée** : 6-12 semaines selon espèce
- **Reprise** : progressive (température, alimentation, introduction)

### Accouplement
- **Introduction** : mâle chez femelle généralement
- **Surveillance** : agressivité possible
- **Durée** : plusieurs accouplements sur semaines

### Gestation/gravité
- **Signes** : gonflement, anorexie, basking prolongé
- **Durée** : variable selon espèce
- **Boîte de ponte** : substrat humide (vermiculite, sphaigne)

### Incubation
- **Substrat** : vermiculite/perlite + eau (ratio 1:1 en poids)
- **Température** : critique (TSD), stable
- **Humidité** : 80-100%, condensation sans contact œufs
- **Incubateur** : commercial ou DIY (thermostat précis)
- **Ne pas retourner** : marquer dessus si nécessaire

### Élevage juvéniles
- **Installations individuelles** : éviter cannibalisme, stress
- **Alimentation fréquente** : croissance rapide
- **Paramètres** : identiques adultes ou légèrement plus chauds
