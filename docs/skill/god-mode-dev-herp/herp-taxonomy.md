# Taxonomie des Reptiles

## Classification Générale

Classe **Reptilia** (paraphylétique traditionnel) / Clade **Sauropsida**

### Ordre SQUAMATA (lézards, serpents, amphisbènes)

#### Sous-ordre Lacertilia (Sauria) - Lézards

**Infraordre Gekkota**
- Gekkonidae : geckos vrais (Eublepharis, Correlophus, Phelsuma, Gekko)
- Eublepharidae : geckos à paupières (Eublepharis macularius)
- Diplodactylidae : geckos australasiens (Rhacodactylus)
- Pygopodidae : geckos apodes australiens

**Infraordre Scincomorpha**
- Scincidae : scinques (Tiliqua, Egernia, Chalcides)
- Lacertidae : lézards vrais (Lacerta, Podarcis, Timon)
- Teiidae : téjus (Salvator, Tupinambis, Ameiva)
- Gymnophthalmidae : microtéiidés

**Infraordre Anguimorpha**
- Varanidae : varans (Varanus)
- Helodermatidae : monstre de Gila, lézard perlé
- Anguidae : orvets, alligator lizards

**Infraordre Iguania**
- Iguanidae : iguanes vrais (Iguana, Cyclura, Ctenosaura)
- Agamidae : agames (Pogona, Uromastyx, Physignathus, Draco)
- Chamaeleonidae : caméléons (Chamaeleo, Furcifer, Trioceros, Brookesia)
- Phrynosomatidae : lézards cornus, fence lizards
- Dactyloidae : anoles (Anolis)

#### Sous-ordre Serpentes - Serpents

**Infraordre Scolecophidia** (serpents fouisseurs primitifs)
- Typhlopidae : serpents aveugles
- Leptotyphlopidae : serpents-fils

**Infraordre Alethinophidia** (serpents vrais)

*Famille Boidae* (boas)
- Boa : Boa constrictor, B. imperator
- Eunectes : anacondas
- Epicrates : boas arc-en-ciel
- Corallus : boas arboricoles (emerald tree boa)
- Acrantophis, Sanzinia : boas malgaches

*Famille Pythonidae* (pythons)
- Python : P. regius, P. molurus, P. reticulatus, P. bivittatus
- Morelia : M. spilota, M. viridis (green tree python)
- Antaresia : pythons nains australiens
- Aspidites : woma, black-headed python

*Famille Colubridae* (couleuvres - paraphylétique)
- Pantherophis : serpents des blés, rat snakes américains
- Lampropeltis : serpents rois, faux-corail
- Elaphe : couleuvres paléarctiques
- Pituophis : serpents-taureaux
- Boaedon : serpents des maisons africains
- Heterodon : serpents à groin

*Famille Natricidae*
- Thamnophis : couleuvres rayées (garter snakes)
- Natrix : couleuvres aquatiques européennes

*Famille Elapidae* (venimeux proteroglyphes)
- Naja : cobras
- Ophiophagus : cobra royal
- Dendroaspis : mambas
- Micrurus : coraux vrais
- Acanthophis : death adders australiens

*Famille Viperidae* (venimeux solenoglyphes)
- Vipera : vipères européennes
- Bitis : vipères africaines (gaboon, puff adder)
- Crotalus, Sistrurus : crotales
- Bothrops : fers de lance
- Agkistrodon : mocassins

*Famille Lamprophiidae*
- Boaedon : serpents des maisons
- Lamprophis : serpents des maisons africains

## Nomenclature & Identification

### Terminologie taxonomique
- Espèce : unité fondamentale (nom binomial)
- Sous-espèce : variation géographique
- Forme/localité : population distincte non décrite
- Phase/morph : variation phénotypique (couleur, pattern)

### Identification morphologique clés
- Écaillure : comptage, disposition (dorsales, ventrales, supralabiales)
- Coloration : patterns, banding, spotting
- Proportions : longueur SVL (snout-vent), queue
- Caractères distinctifs : rostrale, loréale, préoculaire
