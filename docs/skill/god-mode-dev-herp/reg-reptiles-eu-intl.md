# Réglementation Reptiles - Europe & International

## 1. CITES - Convention de Washington

### Présentation
- Convention sur le Commerce International des Espèces Sauvages
- 184 pays signataires
- Réglemente commerce international espèces menacées

### Annexes CITES

#### Annexe I - Protection maximale
- Commerce international interdit (sauf exceptions)
- Espèces menacées d'extinction
- Exemples reptiles :
  - Gavial du Gange
  - Tortues marines
  - Certains crocodiliens
  - Python molure indien (Python molurus molurus)

#### Annexe II - Commerce réglementé
- Commerce autorisé avec permis
- Espèces potentiellement menacées
- Exemples :
  - Majorité des pythons
  - Boas
  - Varans
  - Caméléons
  - Tortues terrestres (Testudo)

#### Annexe III - Protection nationale
- Espèces protégées dans un pays membre
- Commerce avec certificat d'origine

### Documents CITES

#### Permis d'importation/exportation
- Requis pour Annexe I et II hors-UE
- Délivré par autorité nationale (DREAL en France)

#### Certificat d'origine
- Annexe III et certains cas Annexe II

## 2. Réglementation Européenne

### Règlement CE 338/97 (et modifications)
Transposition CITES dans l'UE avec annexes propres :

#### Annexe A (équivalent CITES I + quelques II)
- Protection stricte
- **Certificat intra-communautaire obligatoire** pour tout mouvement
- Exemples :
  - Testudo hermanni, T. graeca, T. marginata
  - Python molurus
  - Uromastyx spp.
  - Nombreux Varanus

#### Annexe B (équivalent CITES II)
- Commerce réglementé
- **Preuve origine légale** requise
- Exemples :
  - Python regius
  - Boa constrictor
  - Morelia spp.
  - Iguana iguana

#### Annexe C (équivalent CITES III)
- Surveillance commerce

#### Annexe D
- Espèces surveillées statistiquement

### Certificats intra-communautaires (CIC)
**Pour espèces Annexe A** :
- Obligatoire pour vente/cession
- Délivré par DREAL/DRIEAT
- Lié à l'individu (marquage)
- Valide dans toute l'UE

### Preuve d'acquisition légale
**Pour Annexe B** :
- Facture éleveur/vendeur
- Registre d'élevage source
- Documentation naissance captivité

## 3. Commerce Intra-UE

### Libre circulation
- Pas de formalités douanières
- MAIS respect réglementations nationales destination

### Espèces Annexe A
- CIC obligatoire accompagne l'animal
- Vérifier validité, correspondance marquage

### Espèces Annexe B et autres
- Facture/attestation cession
- Conformité pays destination

### Points d'attention
- Chaque pays UE peut avoir règles plus strictes
- Vérifier statut espèce dans pays destination
- Certains pays : quarantaine, certificats vétérinaires

## 4. Réglementations Nationales Européennes

### Allemagne (Bundesartenschutzverordnung)
- Liste espèces à déclaration
- Marquage obligatoire tortues Annexe A
- Certificats régionaux variables selon Länder

### Belgique
- Permis d'environnement selon région
- Liste positive (espèces autorisées)
- Certificat capacité pour certaines

### Espagne
- Réglementation par communautés autonomes
- Catalogue espèces exotiques envahissantes
- Trachemys scripta : interdite

### Italie
- Règles strictes espèces dangereuses
- Documentation sanitaire renforcée

### Pays-Bas
- Liste positive espèces autorisées
- Restrictions espèces dangereuses

### Royaume-Uni (post-Brexit)
- Désormais pays tiers (permis CITES requis)
- Dangerous Wild Animals Act
- Licences locales espèces dangereuses

### Suisse (non-UE)
- Convention de Berne
- Permis CITES pour échanges
- Réglementation cantonale variable

## 5. Espèces Envahissantes

### Règlement UE 1143/2014
Liste espèces préoccupation Union :

#### Reptiles concernés
- **Trachemys scripta** (toutes sous-espèces)
  - Interdiction : import, vente, reproduction, relâcher
  - Détention : uniquement spécimens détenus avant interdiction
  - Pas de cession possible

### Implications
- Interdiction commercialisation
- Stérilisation/confinement obligatoire
- Déclaration détention

## 6. Import Hors-UE

### Procédure générale
1. Vérifier statut CITES espèce
2. Obtenir permis export pays origine
3. Demander permis import (DREAL)
4. Point d'entrée désigné (PED)
5. Contrôle douanier + vétérinaire
6. Quarantaine si requise

### Conditions sanitaires
- Règlement UE 2016/429
- Certificat sanitaire pays origine
- Examen vétérinaire possible

### Espèces Annexe I CITES
- Généralement interdit sauf :
  - Nés en captivité génération F2+
  - Élevage enregistré CITES
  - Usage scientifique

### Wild-caught (prélevé nature)
- De plus en plus restreint
- Quotas par pays exportateur
- Traçabilité renforcée
- Certaines espèces : moratoire

## 7. Conventions Internationales

### Convention de Berne (1979)
- Conservation vie sauvage Europe
- Annexes espèces protégées
- Habitats protégés

### Convention de Bonn (CMS)
- Espèces migratrices
- Moins pertinent reptiles terrestres
- Tortues marines concernées

### Protocole de Nagoya
- Accès ressources génétiques
- Partage avantages
- Impact recherche/élevage espèces exotiques

## 9. Conseils Pratiques

### Achat
1. Toujours demander documentation
2. Vérifier correspondance marquage/CIC (Annexe A)
3. Conserver toutes preuves
4. Méfiance prix anormalement bas

### Vente
1. Fournir documentation complète
2. Attestation de cession détaillée
3. Informer obligations acheteur
4. CIC pour Annexe A

### Voyager avec reptiles
1. Vérifier réglementation pays traversés/destination
2. Documents CITES si applicable
3. Certificat sanitaire souvent requis
4. Déclaration douanière

### En cas de contrôle
1. Présenter tous documents
2. Registres à jour
3. Coopérer avec autorités
4. Connaître ses droits
