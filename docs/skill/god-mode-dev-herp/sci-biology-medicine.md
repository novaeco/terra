# Biologie & Médecine

## 1. Biologie Cellulaire

### Cellule procaryote
- Pas de noyau, ADN circulaire
- Ribosomes 70S
- Paroi cellulaire (peptidoglycane)
- Bactéries, archées

### Cellule eucaryote
- Noyau avec membrane
- Organites compartimentés
- Ribosomes 80S
- Animaux, végétaux, champignons, protistes

### Organites
| Organite | Fonction |
|----------|----------|
| Noyau | ADN, transcription |
| Mitochondrie | Respiration, ATP |
| Chloroplaste | Photosynthèse (végétaux) |
| RE rugueux | Synthèse protéines |
| RE lisse | Lipides, détox |
| Golgi | Modification, tri |
| Lysosome | Digestion |
| Peroxisome | Oxydation |
| Cytosquelette | Structure, mouvement |

### Cycle cellulaire
- **Interphase** : G1, S (réplication), G2
- **Mitose** : prophase, métaphase, anaphase, télophase
- **Cytocinèse** : division cytoplasme
- Contrôle : cyclines, CDK, checkpoints

### Méiose
- Division réductionnelle (2n → n)
- Crossing-over, recombinaison
- Gamètes haploïdes
- Diversité génétique

## 2. Génétique

### ADN & gènes
- Double hélice, antiparallèle
- Bases : A-T (2H), G-C (3H)
- Gène : unité d'hérédité
- Génome : ensemble ADN

### Réplication
- Semi-conservative
- Fourche de réplication
- ADN polymérase, primase, ligase
- Fragments d'Okazaki

### Expression génique
- **Transcription** : ADN → ARNm (ARN pol)
- **Traduction** : ARNm → protéine (ribosome)
- Code génétique : codons, 64 → 20 AA
- Régulation : promoteurs, enhancers, facteurs

### Hérédité mendélienne
- Dominant/récessif
- Génotype/phénotype
- Croisements : monohybride, dihybride
- Ségrégation, assortiment indépendant

### Génétique moderne
- Liaison génétique, cartographie
- Épigénétique : méthylation, histones
- Mutations : ponctuelles, chromosomiques
- Biotechnologies : PCR, CRISPR, séquençage

## 5. Anatomie Humaine

### Système squelettique
- 206 os adulte
- Axial : crâne, colonne, côtes
- Appendiculaire : membres, ceintures
- Articulations : synarthroses, amphiarthroses, diarthroses

### Système musculaire
- Squelettique : strié, volontaire
- Cardiaque : strié, involontaire
- Lisse : involontaire, viscères
- Contraction : actine-myosine, ATP

### Système cardiovasculaire
- Cœur : 4 cavités, valves
- Circulation : pulmonaire, systémique
- Vaisseaux : artères, veines, capillaires
- Sang : érythrocytes, leucocytes, plaquettes, plasma

### Système respiratoire
- Voies aériennes : nez, pharynx, larynx, trachée, bronches
- Poumons : alvéoles, échanges gazeux
- Mécanique : diaphragme, muscles intercostaux
- Transport O₂ : hémoglobine

### Système digestif
- Tube digestif : bouche → anus
- Organes annexes : foie, pancréas, vésicule
- Digestion mécanique et chimique
- Absorption intestinale

### Système nerveux
- Central : cerveau, moelle épinière
- Périphérique : nerfs crâniens, spinaux
- Autonome : sympathique, parasympathique
- Neurones : dendrites, axone, synapse, neurotransmetteurs

### Système endocrinien
| Glande | Hormones | Fonctions |
|--------|----------|-----------|
| Hypothalamus | Releasing hormones | Contrôle hypophyse |
| Hypophyse | GH, TSH, ACTH, FSH, LH, PRL | Chef d'orchestre |
| Thyroïde | T3, T4, calcitonine | Métabolisme |
| Parathyroïdes | PTH | Calcium |
| Surrénales | Cortisol, aldostérone, adrénaline | Stress, métabolisme |
| Pancréas | Insuline, glucagon | Glycémie |
| Gonades | Testostérone, œstrogènes, progestérone | Reproduction |

### Système immunitaire
- Inné : barrières, phagocytes, inflammation
- Adaptatif : lymphocytes B (anticorps), T (cellulaire)
- Mémoire immunitaire, vaccination

### Système urinaire
- Reins : filtration, réabsorption, sécrétion
- Néphron : unité fonctionnelle
- Uretères, vessie, urètre
- Régulation : ADH, aldostérone

### Système reproducteur
- Masculin : testicules, épididyme, canal déférent, prostate
- Féminin : ovaires, trompes, utérus, vagin
- Gamétogenèse, fécondation, développement embryonnaire
