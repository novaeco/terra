# Botanique & Dendrologie

## 1. Classification du Vivant Végétal

### Règne Plantae

#### Bryophytes (non-vasculaires)
- Mousses (Bryophyta)
- Hépatiques (Marchantiophyta)
- Anthocérotes (Anthocerotophyta)
- Pas de vrai système vasculaire

#### Ptéridophytes (vasculaires sans graines)
- Fougères (Polypodiopsida)
- Prêles (Equisetopsida)
- Lycopodes (Lycopodiopsida)
- Reproduction par spores

#### Gymnospermes (graines nues)
- **Conifères** (Pinophyta) : pins, sapins, épicéas, cèdres
- **Cycadophytes** : cycas
- **Ginkgophytes** : Ginkgo biloba
- **Gnétophytes** : Ephedra, Welwitschia

#### Angiospermes (plantes à fleurs)
- **Monocotylédones** : graminées, palmiers, orchidées, liliacées
- **Dicotylédones (Eudicots)** : majorité arbres feuillus, légumineuses

## 2. Anatomie Végétale

### Cellule végétale
- Paroi cellulaire (cellulose, lignine)
- Vacuole centrale (turgescence)
- Chloroplastes (photosynthèse)
- Plasmodesmes (communication)

### Tissus végétaux

#### Tissus de revêtement
- Épiderme : protection, stomates, trichomes
- Périderme (liège) : protection secondaire

#### Tissus conducteurs
- **Xylème** : eau + minéraux (ascendant), mort, lignifié
  - Trachéides, éléments de vaisseau
- **Phloème** : sève élaborée (bidirectionnel), vivant
  - Tubes criblés, cellules compagnes

#### Tissus de soutien
- Collenchyme : flexible, vivant
- Sclérenchyme : rigide, mort (fibres, sclérites)

#### Méristèmes
- Apical : croissance longueur
- Latéral (cambium) : croissance diamètre
- Intercalaire : croissance monocots

### Organes

#### Racines
- Absorption eau et minéraux
- Ancrage, stockage
- Types : pivotantes, fasciculées, adventives
- Mycorhizes : symbiose champignons

#### Tiges
- Support, transport, photosynthèse
- Nœuds, entre-nœuds, bourgeons
- Modifications : rhizomes, tubercules, bulbes, stolons

#### Feuilles
- Photosynthèse, échanges gazeux, transpiration
- Limbe, pétiole, nervures
- Types : simples, composées
- Phyllotaxie : alternée, opposée, verticillée

#### Fleurs
- Sépales (calice), pétales (corolle)
- Étamines (androcée) : pollen
- Pistil (gynécée) : ovules
- Types : actinomorphe, zygomorphe

## 3. Physiologie Végétale

### Photosynthèse
```
6 CO₂ + 6 H₂O + lumière → C₆H₁₂O₆ + 6 O₂
```

#### Réactions lumineuses (thylakoïdes)
- Photosystèmes I et II
- Chaîne de transport d'électrons
- ATP + NADPH produits

#### Cycle de Calvin (stroma)
- Fixation CO₂ par RuBisCO
- Réduction → G3P
- Régénération RuBP

#### Variantes photosynthétiques
- **C3** : fixation directe, majorité plantes
- **C4** : préfixation (maïs, canne) - efficace chaleur
- **CAM** : fixation nocturne (cactus, orchidées) - économie eau

### Respiration cellulaire
```
C₆H₁₂O₆ + 6 O₂ → 6 CO₂ + 6 H₂O + ATP
```
- Glycolyse, cycle de Krebs, chaîne respiratoire
- Continue jour et nuit

### Transport
- **Ascendant (xylème)** : transpiration-cohésion-tension
- **Latéral** : symplaste/apoplaste
- **Descendant (phloème)** : pression/flux de masse

### Hormones végétales
| Hormone | Fonctions principales |
|---------|----------------------|
| Auxines | Élongation, tropisme, dominance apicale |
| Cytokinines | Division cellulaire, retard sénescence |
| Gibbérellines | Germination, élongation, floraison |
| Éthylène | Maturation fruits, abscission, sénescence |
| Acide abscissique | Dormance, fermeture stomates, stress |
| Brassinostéroïdes | Croissance, différenciation |

### Tropismes & nasties
- Phototropisme : lumière
- Géotropisme : gravité
- Thigmotropisme : contact
- Hydrotropisme : eau
- Nastie : mouvement non directionnel

## 4. Dendrologie - Science des Arbres

### Croissance des arbres

#### Croissance primaire
- Méristème apical
- Élongation

#### Croissance secondaire
- Cambium vasculaire : bois + liber
- Cambium subéro-phellodermique : écorce
- Cernes annuels (dendrochronologie)

### Anatomie du bois
- **Duramen** : bois de cœur, mort, coloré
- **Aubier** : bois vivant, fonctionnel, clair
- **Cambium** : zone de croissance
- **Liber** : phloème secondaire
- **Écorce** : protection externe

### Identification des arbres

#### Par les feuilles
- Forme : entière, lobée, composée
- Marge : entière, dentée, serratée
- Nervation : pennée, palmée, parallèle
- Disposition : alternée, opposée

#### Par l'écorce
- Lisse, fissurée, écailleuse, exfoliante
- Couleur, texture, motifs

#### Par les fruits/graines
- Samares (érable), glands (chêne), cônes (conifères)
- Drupes, baies, capsules

## 5. Familles d'Arbres Importantes

### Conifères

#### Pinaceae
- **Pinus** (pins) : 2-5 aiguilles par fascicule
- **Picea** (épicéas) : aiguilles isolées, piquantes
- **Abies** (sapins) : aiguilles plates, ventouses
- **Larix** (mélèzes) : caducs
- **Cedrus** (cèdres) : aiguilles en rosettes

#### Cupressaceae
- **Cupressus** (cyprès) : écailles, méditerranéen
- **Juniperus** (genévriers) : baies
- **Thuja** : écailles aplaties
- **Sequoia, Sequoiadendron** : géants

#### Taxaceae
- **Taxus** (ifs) : toxique, arille rouge

### Feuillus

#### Fagaceae
- **Quercus** (chênes) : glands, 600+ espèces
- **Fagus** (hêtres) : feuilles entières, faines
- **Castanea** (châtaigniers) : châtaignes

#### Betulaceae
- **Betula** (bouleaux) : écorce blanche
- **Alnus** (aulnes) : zones humides, fixation N
- **Carpinus** (charmes) : bois dur
- **Corylus** (noisetiers) : noisettes

#### Salicaceae
- **Salix** (saules) : zones humides, flexibles
- **Populus** (peupliers) : croissance rapide

#### Aceraceae / Sapindaceae
- **Acer** (érables) : feuilles palmées, samares

#### Oleaceae
- **Fraxinus** (frênes) : feuilles composées
- **Olea** (oliviers) : méditerranéen

#### Rosaceae
- **Prunus** : cerisiers, pruniers, amandiers
- **Malus** : pommiers
- **Sorbus** : sorbiers, alisiers

#### Fabaceae
- **Robinia** (robiniers) : fixation N, épines
- **Acacia** : tropicaux, savanes

## 6. Écologie Forestière

### Strates forestières
- Strate arborée (canopée)
- Strate arbustive
- Strate herbacée
- Strate muscinale
- Litière

### Successions écologiques
- Pionnières : bouleaux, pins
- Post-pionnières : chênes, hêtres
- Climax : forêt mature stable

### Interactions
- Compétition : lumière, eau, nutriments
- Facilitation : amélioration conditions
- Mycorhizes : 90% plantes terrestres
- Réseau mycorhizien : "Wood Wide Web"

### Biomes forestiers
- Forêt boréale (taïga) : conifères, froid
- Forêt tempérée : feuillus caducs
- Forêt méditerranéenne : sclérophylle
- Forêt tropicale humide : biodiversité max
- Forêt tropicale sèche : caducifoliée

## 8. Plantes Médicinales & Toxiques

### Principes actifs
- Alcaloïdes : morphine, caféine, nicotine
- Glycosides : digitaline, saponines
- Terpènes : menthol, camphre
- Flavonoïdes : antioxydants

### Plantes médicinales courantes
- Camomille, menthe, thym, lavande
- Échinacée, millepertuis, valériane
- Ginkgo, ginseng, aloe vera

### Plantes toxiques
- If (taxine) : mortel
- Digitale (digitaline) : cardiaque
- Aconit : très toxique
- Belladone, datura : anticholinergiques
- Ricin : toxine mortelle

## 9. Conservation & Menaces

### Menaces
- Déforestation, fragmentation
- Changement climatique
- Espèces invasives
- Maladies (chalarose, graphiose)
- Surexploitation

### Conservation
- Aires protégées, parcs nationaux
- Banques de graines (Svalbard)
- Restauration écologique
- Gestion forestière durable (FSC, PEFC)
