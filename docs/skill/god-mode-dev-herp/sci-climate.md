# Climatologie & Météorologie

## 1. Atmosphère Terrestre

### Structure verticale
- **Troposphère** (0-12km) : météo, 75% masse atm, T° décroît (-6.5°C/km)
- **Stratosphère** (12-50km) : couche d'ozone, T° croît
- **Mésosphère** (50-80km) : T° décroît, étoiles filantes
- **Thermosphère** (80-700km) : T° très élevée, aurores
- **Exosphère** (>700km) : transition vers espace

### Composition
- N₂ (78%), O₂ (21%), Ar (0.93%), CO₂ (0.04%)
- Vapeur d'eau : variable (0-4%)
- Gaz traces : CH₄, N₂O, O₃, CFCs

## 2. Rayonnement & Bilan Énergétique

### Rayonnement solaire
- Constante solaire : ~1361 W/m²
- Spectre : UV (7%), visible (43%), IR (50%)
- Albédo terrestre moyen : ~0.30

### Effet de serre
- GES : H₂O, CO₂, CH₄, N₂O, O₃, CFCs
- Forçage radiatif : W/m² supplémentaires
- Sensibilité climatique : ΔT pour doublement CO₂ (~3°C)

### Bilan radiatif
- Entrant solaire : 340 W/m²
- Réfléchi : 100 W/m²
- Absorbé surface : 160 W/m²
- Absorbé atmosphère : 80 W/m²

## 3. Circulation Atmosphérique

### Cellules de circulation
- **Hadley** (0-30°) : ascendance équateur, subsidence subtropicale
- **Ferrel** (30-60°) : circulation indirecte, tempérée
- **Polaire** (60-90°) : air froid dense

### Vents dominants
- Alizés : NE (hém. nord), SE (hém. sud)
- Westerlies : vents d'ouest moyennes latitudes
- Easterlies polaires

### Jet streams
- Jet polaire : 9-12km, 30-60°, >100 km/h
- Jet subtropical : 10-16km, ~30°
- Méandres, blocages, influence météo

### Mousson
- Renversement saisonnier circulation
- Gradient thermique terre-océan
- Mousson asiatique, africaine, américaine

## 4. Océanographie Climatique

### Circulation thermohaline
- Convection profonde : Atlantique Nord, Antarctique
- AMOC (Atlantic Meridional Overturning Circulation)
- Temps de circulation : ~1000 ans
- Transport chaleur vers pôles

### Courants de surface
- Gulf Stream, Kuroshio (chauds)
- Courants de Humboldt, Benguela (froids)
- Gyres océaniques

### Interactions océan-atmosphère
- **ENSO** (El Niño/La Niña) : oscillation Pacifique
- **NAO** : oscillation Nord-Atlantique
- **PDO** : oscillation décennale Pacifique
- **AMO** : oscillation multidécennale Atlantique

## 5. Phénomènes Météorologiques

### Formation nuages
- Condensation : T° < point de rosée
- Noyaux de condensation (CCN)
- Types : cumulus, stratus, cirrus, nimbus

### Classification nuages
| Étage | Types | Altitude |
|-------|-------|----------|
| Haut | Cirrus, Cirrostratus, Cirrocumulus | >6km |
| Moyen | Altostratus, Altocumulus | 2-6km |
| Bas | Stratus, Stratocumulus, Nimbostratus | <2km |
| Développement vertical | Cumulus, Cumulonimbus | variable |

### Précipitations
- Processus Bergeron : cristaux de glace
- Coalescence : collision gouttelettes
- Types : pluie, neige, grêle, verglas

### Fronts & dépressions
- Front chaud : air chaud glisse sur froid
- Front froid : air froid soulève chaud
- Front occlus : fusion fronts
- Cyclogenèse : formation dépressions

### Phénomènes violents
- **Orages** : cumulonimbus, instabilité, CAPE
- **Tornades** : supercellules, échelle Fujita
- **Cyclones tropicaux** : T° océan >26°C, Coriolis
- **Tempêtes** : dépressions intenses moyennes latitudes

## 6. Climatologie Régionale

### Zones climatiques (Köppen)
- **A** - Tropical : Af (équatorial), Am (mousson), Aw (savane)
- **B** - Aride : BWh (désert chaud), BWk (désert froid), BSh, BSk
- **C** - Tempéré : Cfa, Cfb, Csa, Csb, Cwa
- **D** - Continental : Dfa, Dfb, Dfc, Dfd, Dwa, Dwb
- **E** - Polaire : ET (toundra), EF (glacial)

### Microclimats
- Effet urbain (îlot de chaleur)
- Relief : exposition, altitude, effet de foehn
- Proximité eau : effet tampon
- Végétation : évapotranspiration

## 7. Changement Climatique

### Observations actuelles
- T° moyenne : +1.1°C depuis préindustriel
- Niveau mer : +3.7 mm/an (accélération)
- Banquise arctique : -13%/décennie (septembre)
- Glaciers : recul généralisé
- Événements extrêmes : intensification

### Causes anthropiques
- CO₂ : 280 → 420+ ppm
- CH₄ : 700 → 1900 ppb
- Déforestation, agriculture, industrie, transport
- Attribution : >95% confiance origine humaine

### Projections GIEC (IPCC)
| Scénario | T° 2100 | Description |
|----------|---------|-------------|
| SSP1-1.9 | +1.4°C | Durabilité, net-zero 2050 |
| SSP1-2.6 | +1.8°C | Durabilité modérée |
| SSP2-4.5 | +2.7°C | Trajectoire actuelle |
| SSP3-7.0 | +3.6°C | Rivalités régionales |
| SSP5-8.5 | +4.4°C | Développement fossile |

### Points de basculement (tipping points)
- Fonte Groenland, Antarctique Ouest
- Dégel pergélisol (méthane)
- Ralentissement AMOC
- Dépérissement Amazonie
- Disparition récifs coralliens

### Adaptation & atténuation
- Réduction émissions, énergies renouvelables
- Capture carbone, reforestation
- Infrastructure résiliente
- Migration espèces, agriculture adaptée

## 8. Paléoclimatologie

### Proxies climatiques
- Carottes glace : isotopes O¹⁸, bulles d'air
- Sédiments marins : foraminifères
- Dendrochronologie : cernes arbres
- Spéléothèmes : stalagmites
- Pollens : reconstitution végétation

### Périodes clés
- **Quaternaire** : cycles glaciaires (Milankovitch)
- **Éémien** (-130-115 ka) : interglaciaire chaud
- **Dernier maximum glaciaire** (-21 ka) : -5°C
- **Holocène** (-11.7 ka - présent) : stable
- **Optimum médiéval, petit âge glaciaire**

### Cycles Milankovitch
- Excentricité : 100 000 ans
- Obliquité : 41 000 ans
- Précession : 23 000 ans

## 9. Modélisation Climatique

### Types de modèles
- **GCM** : General Circulation Model
- **ESM** : Earth System Model (biogéochimie)
- **RCM** : Regional Climate Model
- **EBM** : Energy Balance Model (simple)

### Composantes
- Atmosphère, océan, glace, surface
- Couplage, rétroactions
- Résolution spatiale/temporelle
- Paramétrisations sous-maille

### Validation
- Réanalyses (ERA5, MERRA)
- Comparaison observations
- Ensembles multi-modèles
- Incertitudes quantifiées

## 10. Instrumentation & Observation

### Mesures in-situ
- Stations météo : T°, P, humidité, vent
- Radiosondes : profils verticaux
- Bouées océaniques, Argo floats
- Tours de flux (CO₂, énergie)

### Télédétection
- Satellites géostationnaires : imagerie continue
- Satellites polaires : couverture globale
- Radar, lidar : précipitations, aérosols
- Altimétrie : niveau mer, glaces

### Réseaux
- OMM (WMO) : coordination mondiale
- GCOS : système observation climat
- Copernicus, NOAA, NASA, Météo-France
