# Entomologie - Science des Insectes

## 1. Classification des Insectes

### Position phylogénétique
- Arthropodes > Hexapodes > Insectes
- ~1 million d'espèces décrites, ~5-10 millions estimées
- 80% des espèces animales connues

### Ordres principaux

#### Aptérygotes (primitifs, sans ailes)
- **Archaeognatha** : machilides
- **Zygentoma** : lépismes (poissons d'argent)

#### Ptérygotes (ailés ou secondairement aptères)

**Paléoptères** (ailes non repliables)
- **Ephemeroptera** : éphémères
- **Odonata** : libellules, demoiselles

**Néoptères** (ailes repliables)

*Polyneoptera*
- **Blattodea** : blattes, termites
- **Mantodea** : mantes
- **Orthoptera** : criquets, sauterelles, grillons
- **Phasmatodea** : phasmes
- **Dermaptera** : perce-oreilles

*Paraneoptera*
- **Hemiptera** : punaises, cigales, pucerons, cochenilles
- **Thysanoptera** : thrips
- **Psocodea** : poux

*Holometabola* (métamorphose complète)
- **Coleoptera** : coléoptères (scarabées, coccinelles, charançons)
- **Lepidoptera** : papillons, mites
- **Diptera** : mouches, moustiques
- **Hymenoptera** : abeilles, guêpes, fourmis
- **Neuroptera** : chrysopes, fourmilions
- **Siphonaptera** : puces
- **Trichoptera** : phryganes

## 2. Anatomie des Insectes

### Exosquelette
- Cuticule : chitine + protéines
- Sclérites, sutures
- Mues (ecdysis), croissance discontinue

### Tagmes (régions du corps)

#### Tête
- Antennes : sensorielles (olfaction, toucher)
- Yeux composés : ommatidies
- Ocelles : yeux simples
- Pièces buccales :
  - Broyeur : mandibules (orthoptères, coléoptères)
  - Piqueur-suceur : stylets (hémiptères, diptères)
  - Lécheur-suceur : trompe (lépidoptères)
  - Spongieux : labelle (mouches)

#### Thorax
- 3 segments : pro-, méso-, métathorax
- 3 paires de pattes (hexapodes)
- 2 paires d'ailes (ptérygotes) ou modifiées/absentes
- Modifications ailes : élytres (coléoptères), haltères (diptères)

#### Abdomen
- 11 segments max (souvent moins visibles)
- Organes reproducteurs, cerques
- Pas d'appendices locomoteurs

### Systèmes internes
- **Respiratoire** : trachées, spiracles (pas de poumons)
- **Circulatoire** : ouvert, hémolymphe, vaisseau dorsal
- **Digestif** : tube digestif, tubes de Malpighi (excrétion)
- **Nerveux** : ganglions, cerveau tripartite

## 3. Développement

### Métamorphose incomplète (hémimétaboles)
- Œuf → Nymphes → Adulte
- Nymphes ressemblent aux adultes
- Orthoptères, hémiptères, odonates

### Métamorphose complète (holométaboles)
- Œuf → Larve → Nymphe (pupe) → Adulte
- Larve très différente de l'adulte
- Coléoptères, lépidoptères, diptères, hyménoptères

### Types de larves
- Campodéiforme : active, prédatrice
- Éruciforme : chenilles
- Vermiforme : asticots
- Scarabéiforme : vers blancs

## 4. Physiologie

### Respiration
- Système trachéen : ramifications jusqu'aux cellules
- Spiracles : ouvertures contrôlées
- Limite de taille des insectes

### Thermorégulation
- Ectothermes mais comportement thermorégulateur
- Frisson thoracique (vol)
- Coloration, exposition solaire

### Communication
- **Phéromones** : sexuelles, d'alarme, de piste
- **Stridulation** : frottement (orthoptères)
- **Bioluminescence** : lucioles
- **Visuelle** : couleurs, danses

### Vol
- Muscles directs ou indirects
- Fréquence : 5 Hz (papillons) à 1000 Hz (moucherons)
- Mécanismes : asynchrone, synchrone

## 5. Écologie des Insectes

### Rôles écologiques
- **Pollinisation** : abeilles, papillons, mouches
- **Décomposition** : bousiers, mouches, termites
- **Chaînes alimentaires** : proies essentielles
- **Prédation** : libellules, mantes, coccinelles
- **Parasitisme/parasitoïdes** : guêpes parasitoïdes

### Insectes sociaux
- **Hyménoptères** : abeilles, fourmis, guêpes
- **Isoptères** : termites
- Castes : reines, ouvrières, soldats, mâles
- Communication : phéromones, danse (abeilles)
- Superorganisme

### Insectes nuisibles
- **Ravageurs agricoles** : criquets, pucerons, charançons
- **Vecteurs de maladies** : moustiques (paludisme, dengue), mouches tsé-tsé
- **Parasites** : poux, puces
- **Domestiques** : blattes, mites

### Insectes utiles
- Pollinisateurs (valeur économique énorme)
- Ennemis naturels (lutte biologique)
- Produits : miel, soie, cire, laque
- Alimentation humaine (entomophagie)

## 6. Insectes d'Élevage (pour reptiles)

### Grillons (Acheta domesticus, Gryllus spp.)
- Température : 25-30°C
- Alimentation : fruits, légumes, céréales, protéines
- Cycle : 6-8 semaines
- Gut-loading avant distribution

### Criquets/Locustes (Locusta migratoria, Schistocerca)
- Température : 28-35°C
- Alimentation : herbes, céréales
- Plus nutritifs que grillons
- Attention aux évasions

### Blattes
- **Blaptica dubia** : ne grimpe pas, ne vole pas
- **Nauphoeta cinerea** : facile, prolifique
- **Gromphadorhina portentosa** : Madagascar, grande taille
- Température : 25-32°C selon espèce

### Vers de farine (Tenebrio molitor)
- Larves de coléoptère
- Facile, longue conservation
- Riche en lipides (modération)
- Exosquelette chitineux

### Vers morios (Zophobas morio)
- Plus gros que vers de farine
- Ne pas donner aux juvéniles (morsures)
- Même élevage que Tenebrio

### Vers à soie (Bombyx mori)
- Très nutritifs, mous
- Élevage sur feuilles de mûrier
- Ratio Ca:P favorable

### Mouches soldats noires (Hermetia illucens)
- Larves très nutritives
- Élevage sur déchets organiques
- Excellentes pour reptiles

### Drosophiles
- Pour petits reptiles, amphibiens
- Souches aptères disponibles
- Cycle rapide

## 7. Entomologie Appliquée

### Lutte biologique
- Parasitoïdes (Trichogramma)
- Prédateurs (coccinelles)
- Pathogènes (Bt, champignons)

### Entomologie légale
- Succession d'insectes nécrophages
- Datation des décès
- Calliphoridae, Sarcophagidae

### Apiculture
- Apis mellifera
- Ruches, essaimage
- Produits : miel, cire, propolis, gelée royale
- Déclin des pollinisateurs

### Sériciculture
- Bombyx mori
- Production de soie
- Asie principalement
