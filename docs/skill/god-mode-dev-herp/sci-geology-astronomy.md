# Géologie & Astronomie

## 1. Structure de la Terre

### Couches internes
| Couche | Profondeur | Composition | État |
|--------|------------|-------------|------|
| Croûte continentale | 0-70 km | Granites, roches sédimentaires | Solide |
| Croûte océanique | 0-10 km | Basaltes | Solide |
| Manteau supérieur | 10-410 km | Péridotites | Solide/ductile |
| Zone de transition | 410-660 km | Minéraux HP | Solide |
| Manteau inférieur | 660-2900 km | Silicates Mg-Fe | Solide |
| Noyau externe | 2900-5150 km | Fe-Ni | Liquide |
| Noyau interne | 5150-6371 km | Fe-Ni | Solide |

### Discontinuités
- Mohorovičić (Moho) : croûte/manteau
- Gutenberg : manteau/noyau
- Lehmann : noyau externe/interne

## 2. Tectonique des Plaques

### Théorie
- Lithosphère divisée en plaques rigides
- Asthénosphère ductile dessous
- Mouvements : cm/an

### Types de limites
- **Divergentes** : dorsales, extension, volcans
- **Convergentes** : subduction, collision, fosses
- **Transformantes** : coulissage, failles

### Moteur
- Convection mantellique
- Poussée aux dorsales
- Traction des plaques subductées

### Conséquences
- Séismes, volcans
- Formation montagnes
- Ouverture/fermeture océans
- Dérive des continents

## 3. Minéralogie

### Classification des minéraux
- Silicates (90% croûte) : quartz, feldspaths, micas, olivine
- Carbonates : calcite, dolomite
- Oxydes : hématite, magnétite
- Sulfures : pyrite, galène
- Halogénures : halite, fluorite
- Sulfates : gypse
- Natifs : or, cuivre, diamant

### Propriétés
- Dureté (Mohs 1-10)
- Éclat, couleur, trace
- Clivage, cassure
- Densité, système cristallin

## 4. Pétrologie

### Roches ignées
- **Plutoniques** (refroidissement lent) : granite, gabbro, diorite
- **Volcaniques** (refroidissement rapide) : basalte, rhyolite, andésite
- Classification : % silice, texture

### Roches sédimentaires
- **Détritiques** : grès, argile, conglomérat
- **Chimiques** : calcaire, évaporites
- **Biogènes** : craie, charbon, pétrole
- Processus : altération → transport → dépôt → diagenèse

### Roches métamorphiques
- Transformation P-T sans fusion
- **Contact** : hornfels
- **Régional** : schiste, gneiss, marbre
- Faciès métamorphiques

### Cycle des roches
Ignée → (altération) → Sédimentaire → (métamorphisme) → Métamorphique → (fusion) → Ignée

## 9. Étoiles & Galaxies

### Classification stellaire
- OBAFGKM (température décroissante)
- Diagramme HR : luminosité vs température
- Séquence principale, géantes, naines

### Évolution stellaire
1. Nuage moléculaire → protoétoile
2. Séquence principale (fusion H)
3. Géante rouge (fusion He)
4. Fin de vie selon masse :
   - <8 M☉ : naine blanche
   - 8-25 M☉ : supernova → étoile à neutrons
   - >25 M☉ : supernova → trou noir

### Galaxies
- Types : elliptiques, spirales, irrégulières
- Voie Lactée : spirale barrée, ~200 milliards étoiles
- Groupe Local, amas, superamas

### Cosmologie
- Big Bang : 13.8 Ga
- Expansion univers (Hubble)
- Matière noire, énergie sombre
- CMB : fond diffus cosmologique
