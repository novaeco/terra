# Physique & Chimie

## 1. Mécanique Classique

### Cinématique
- Position, vitesse, accélération
- MRU, MRUA, mouvement circulaire
- Équations horaires, trajectoires

### Dynamique (Lois de Newton)
1. Inertie : ΣF = 0 → v = cte
2. F = ma
3. Action-réaction : F₁₂ = -F₂₁

### Énergie & travail
- W = F·d·cos(θ)
- Ec = ½mv², Ep = mgh
- Conservation énergie mécanique
- Puissance : P = W/t = F·v

### Quantité de mouvement
- p = mv
- Conservation : Σp = cte (système isolé)
- Collisions élastiques, inélastiques

### Rotation
- θ, ω, α (angulaires)
- Moment d'inertie : I = Σmr²
- Moment cinétique : L = Iω
- Couple : τ = r × F

### Gravitation
- F = Gm₁m₂/r²
- Lois de Kepler
- Orbites, satellites, énergie orbitale

## 2. Électromagnétisme

### Électrostatique
- Loi de Coulomb : F = kq₁q₂/r²
- Champ électrique : E = F/q
- Potentiel : V = kq/r
- Théorème de Gauss

### Courant électrique
- I = dq/dt
- Loi d'Ohm : V = RI
- Puissance : P = VI = RI² = V²/R
- Circuits série/parallèle

### Magnétisme
- Champ magnétique B (Tesla)
- Force de Lorentz : F = qv × B
- Loi de Biot-Savart
- Théorème d'Ampère

### Induction
- Loi de Faraday : ε = -dΦ/dt
- Loi de Lenz
- Inductance, transformateurs
- Courants de Foucault

### Équations de Maxwell
1. ∇·E = ρ/ε₀ (Gauss électrique)
2. ∇·B = 0 (pas de monopôle)
3. ∇×E = -∂B/∂t (Faraday)
4. ∇×B = μ₀J + μ₀ε₀∂E/∂t (Ampère-Maxwell)

### Ondes électromagnétiques
- c = 1/√(μ₀ε₀) = 3×10⁸ m/s
- Spectre : radio → micro-ondes → IR → visible → UV → X → γ

## 3. Thermodynamique

### Concepts fondamentaux
- Température, chaleur, travail
- Équilibre thermique (loi zéro)
- Gaz parfait : PV = nRT

### Premier principe
- ΔU = Q - W
- Conservation de l'énergie
- Enthalpie : H = U + PV

### Deuxième principe
- Entropie : S, ΔS ≥ 0 (univers)
- Irréversibilité
- Rendement Carnot : η = 1 - Tc/Th

### Troisième principe
- S → 0 quand T → 0 K

### Transformations
- Isotherme : T = cte
- Isobare : P = cte
- Isochore : V = cte
- Adiabatique : Q = 0

### Machines thermiques
- Moteurs, réfrigérateurs, pompes à chaleur
- Cycles : Carnot, Otto, Diesel, Rankine
