# Zoologie Complète

## 1. Classification du Règne Animal

### Arbre phylogénétique simplifié
```
Animalia
├── Parazoa (éponges)
├── Eumetazoa
│   ├── Radiata (cnidaires, cténaires)
│   └── Bilateria
│       ├── Protostomia
│       │   ├── Ecdysozoa (arthropodes, nématodes)
│       │   └── Lophotrochozoa (mollusques, annélides)
│       └── Deuterostomia
│           ├── Echinodermata
│           └── Chordata (vertébrés)
```

## 2. Invertébrés

### Porifères (éponges)
- Organisation cellulaire simple
- Choanocytes, spicules
- Filtration, régénération

### Cnidaires
- **Hydrozoaires** : hydres, physalies
- **Scyphozoaires** : méduses vraies
- **Anthozoaires** : coraux, anémones
- Cnidocytes, cycle métagénétique

### Mollusques
- **Gastéropodes** : escargots, limaces, nudibranches
- **Bivalves** : moules, huîtres, palourdes
- **Céphalopodes** : poulpes, calmars, seiches
- Manteau, coquille, radula

### Annélides
- **Polychètes** : vers marins
- **Oligochètes** : lombrics
- **Hirudinées** : sangsues
- Métamérie, soies

### Arthropodes
Voir référence entomologie pour insectes

#### Crustacés
- Décapodes : crabes, crevettes, homards
- Isopodes : cloportes
- Copépodes, cirripèdes (balanes)

#### Arachnides
- Araignées : chélicères, filières
- Scorpions : pédipalpes, telson
- Acariens, tiques
- Opilions, solifuges

#### Myriapodes
- Chilopodes (centipèdes) : carnivores
- Diplopodes (millipèdes) : détritivores

### Échinodermes
- Étoiles de mer, oursins, holothuries
- Symétrie pentaradiée, système aquifère
- Régénération

### Nématodes
- Vers ronds, ubiquistes
- Parasites (ascaris, oxyures) ou libres
- Cuticule, mues

## 3. Poissons

### Chondrichtyens (cartilagineux)
- **Requins** : détection électrique, dents renouvelables
- **Raies** : benthiques, nageoires pectorales
- **Chimères**
- Pas de vessie natatoire, foie lipidique

### Ostéichtyens (osseux)

#### Actinoptérygiens (nageoires rayonnées)
- 99% des poissons
- Téléostéens : perches, truites, thons, poissons-clowns
- Vessie natatoire, écailles cycloïdes/cténoïdes

#### Sarcoptérygiens (nageoires charnues)
- Cœlacanthes, dipneustes
- Ancêtres des tétrapodes

### Biologie des poissons
- Respiration branchiale, opercule
- Ligne latérale (mécanoréception)
- Osmorégulation : eau douce vs marine
- Reproduction : ovipares, ovovivipares, vivipares

### Poissons d'aquarium
- Eau douce tropicale : guppys, néons, scalaires, discus, cichlidés
- Eau froide : poissons rouges, carpes koï
- Marins : poissons-clowns, chirurgiens, demoiselles

## 4. Amphibiens

### Anoures (grenouilles, crapauds)
- Métamorphose : têtard aquatique → adulte
- Peau perméable, respiration cutanée
- Chant nuptial, amplexus

### Urodèles (salamandres, tritons)
- Queue conservée adulte
- Néoténie (axolotl)
- Régénération membres

### Gymnophiones (cécilies)
- Apodes, fouisseurs
- Tropicaux

### Biologie amphibienne
- Ectothermie
- Peau glandulaire (toxines)
- Déclin mondial : chytridiomycose

## 5. Oiseaux

### Classification
- **Paléognathes** : autruches, émeus, kiwis
- **Néognathes** : majorité
  - Passériformes (60%) : moineaux, corbeaux, merles
  - Psittaciformes : perroquets
  - Falconiformes, Accipitriformes : rapaces
  - Strigiformes : hiboux, chouettes
  - Anseriformes : canards, oies
  - Galliformes : poules, faisans
  - Columbiformes : pigeons
  - Piciformes : pics

### Anatomie
- Plumes : vol, isolation, parade
- Os pneumatiques, bréchet
- Système respiratoire : sacs aériens, flux unidirectionnel
- Gésier, absence de dents

### Biologie
- Endothermie, métabolisme élevé
- Vol : battement, plané, vol stationnaire
- Migration : navigation magnétique, solaire
- Chant : syrinx, territorialité, parade

### Reproduction
- Ovipares exclusifs
- Incubation, soins parentaux
- Nidifuges vs nidicoles

## 6. Mammifères

### Monotrèmes
- Ovipares : ornithorynque, échidnés
- Australasie
- Électroréception (ornithorynque)

### Marsupiaux
- Poche marsupiale, naissance prématurée
- Kangourous, koalas, opossums, wombats
- Australasie, Amérique

### Placentaires (Euthériens)

#### Afrothériens
- Éléphants, damans, lamantins, oryctéropes

#### Xénarthres
- Tatous, paresseux, fourmiliers

#### Laurasiathériens
- **Carnivores** : félins, canidés, ursidés, mustélidés
- **Périssodactyles** : chevaux, rhinocéros, tapirs
- **Artiodactyles** : bovins, cervidés, suidés, camélidés
- **Cétacés** : baleines, dauphins
- **Chiroptères** : chauves-souris (écholocation)
- **Eulipotyphles** : hérissons, taupes, musaraignes

#### Euarchontoglires
- **Primates** : lémuriens, singes, grands singes, humains
- **Rongeurs** : souris, rats, écureuils, castors
- **Lagomorphes** : lapins, lièvres

### Biologie mammalienne
- Endothermie, poils, glandes sudoripares
- Glandes mammaires, allaitement
- Placenta (sauf monotrèmes)
- Cerveau développé, comportements complexes
