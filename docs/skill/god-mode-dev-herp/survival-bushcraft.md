# Survie & Bushcraft

## 1. Priorités de Survie (Règle des 3)

### Hiérarchie des besoins
- **3 minutes** sans air
- **3 heures** sans abri (conditions extrêmes)
- **3 jours** sans eau
- **3 semaines** sans nourriture

### Acronyme SURVIVAL
- **S**ize up the situation (évaluer)
- **U**ndue haste makes waste (ne pas précipiter)
- **R**emember where you are (s'orienter)
- **V**anquish fear and panic (contrôler peur)
- **I**mprovise and improve (improviser)
- **V**alue living (volonté de vivre)
- **A**ct like the natives (observer environnement)
- **L**earn basic skills (compétences de base)

## 2. Abri

### Types d'abris naturels
- **Lean-to** : appentis simple, rapide
- **Debris hut** : isolation débris végétaux
- **Tarp shelter** : bâche, configurations multiples
- **Snow shelter** : quinzee, igloo, tranchée

### Principes
- Protection vent, pluie, froid/chaleur
- Isolation du sol (branches, feuilles)
- Taille minimale (conserver chaleur)
- Orientation : dos au vent dominant
- Près d'eau mais pas zone inondable

### Construction
- Charpente solide avant couverture
- Couches multiples d'isolation
- Entrée petite, orientée
- Réflecteur de chaleur (feu devant)

## 3. Eau

### Besoins
- Minimum 2L/jour (plus si effort, chaleur)
- Déshydratation : fatigue, confusion, danger

### Sources
- Cours d'eau, lacs
- Eau de pluie (collecte)
- Rosée (tissu absorbant)
- Neige/glace (fondre, ne pas manger directement)
- Plantes : vigne, bambou, cactus (attention toxicité)
- Condensation solaire (still)

### Purification
| Méthode | Efficacité | Notes |
|---------|------------|-------|
| Ébullition | Excellente | 1-3 min, plus en altitude |
| Filtration | Bonne | Particules, certains pathogènes |
| Chimique (iode, chlore) | Bonne | Goût, temps d'action |
| UV (SteriPen) | Excellente | Batteries nécessaires |
| Filtres pompe | Excellente | Poids, entretien |

### Signes déshydratation
- Soif, bouche sèche
- Urine foncée, peu fréquente
- Fatigue, maux de tête
- Confusion (stade avancé)

## 4. Feu

### Méthodes d'allumage
- **Allumettes tempête** : fiables, imperméables
- **Briquet** : pratique, altitude difficile
- **Firesteel (ferrocerium)** : durable, toutes conditions
- **Friction** : archet (bow drill), main (hand drill)
- **Loupe/miroir** : soleil nécessaire

### Préparation
1. **Amadou** (tinder) : fibres sèches, écorce bouleau, champignon amadouvier
2. **Petit bois** (kindling) : brindilles, copeaux
3. **Bois de chauffe** (fuel) : branches croissantes
4. Plateforme sèche

### Configuration feux
- **Tipi** : allumage rapide
- **Log cabin** : stable, durable
- **Long fire** : chaleur pour dormir
- **Star fire** : économe en bois
- **Dakota hole** : discret, efficace

### Sécurité
- Zone dégagée, coupe-feu
- Vent modéré
- Extinction complète (noyé + dispersé)

## 5. Nourriture

### Priorités
- Eau > Abri > Feu > Nourriture
- Jeûne court (jours) : gérable
- Effort de chasse/cueillette vs calories

### Cueillette
- **Règle de base** : ne manger que ce qu'on identifie 100%
- Test universel d'edibilité (8h, risqué)
- Plantes comestibles courantes : ortie, pissenlit, plantain, ail des ours
- Éviter : champignons inconnus, baies blanches/jaunes

### Insectes
- Source protéines accessible
- Comestibles : vers, larves, criquets, fourmis
- Cuire de préférence
- Éviter : couleurs vives, poilus, odorants

### Piégeage
- Collets : petit gibier
- Pièges à assommoir (deadfall)
- Pièges à poisson
- Légalité : situations d'urgence uniquement

### Pêche
- Hameçons improvisés (os, épines, bois dur)
- Ligne : fibres naturelles, paracord intérieur
- Nasses, barrages
- Lances, arcs pêche

### Chasse
- Arc improvisé (long apprentissage)
- Lance, bâton de lancer
- Fronde

### Conservation
- Fumage : préservation longue
- Séchage : viande séchée (jerky)
- Sel (si disponible)

## 6. Navigation

### Orientation sans boussole
- **Soleil** : est le matin, ouest le soir
- **Étoiles** : étoile polaire (nord), Croix du Sud
- **Mousse** : indicateur peu fiable
- **Ombres** : méthode du bâton (est-ouest)

### Avec boussole
- Déclinaison magnétique
- Azimut, relèvement
- Triangulation

### Cartes
- Lecture topographique
- Courbes de niveau
- Échelle, légende

### Signalisation
- Signal visuel : miroir, fumée, SOS
- Signal sonore : sifflet (3 coups = détresse)
- Marquage : pierres, branches, traces

## 7. Premiers Secours Survie

### Priorités (ABC)
- **Airway** : voies respiratoires
- **Breathing** : respiration
- **Circulation** : pouls, hémorragies

### Hémorragies
- Pression directe
- Élévation
- Garrot (dernier recours)
- Packing plaie

### Fractures
- Immobilisation
- Attelle improvisée
- Ne pas réduire

### Hypothermie
- Signes : frissons, confusion, léthargie
- Réchauffement progressif
- Vêtements secs, abri, chaleur

### Hyperthermie
- Coup de chaleur : urgence vitale
- Ombre, hydratation, refroidissement

### Morsures/Piqûres
- Serpents : immobilisation, évacuation
- Insectes : retirer dard, antihistaminiques
- Tiques : retrait complet, surveillance

## 8. Équipement

### Kit de survie minimal (EDC)
- Couteau fiable
- Firesteel + amadou
- Boussole
- Sifflet
- Fil/cordage
- Sac poubelle (abri urgence)

### Kit complet (72h)
- Eau : gourde, purification
- Abri : bâche, couverture survie
- Feu : allumettes, briquet, firesteel
- Nourriture : rations, kit pêche
- Premiers soins
- Navigation : carte, boussole
- Signalisation : miroir, sifflet
- Outils : couteau, scie, cordage

### Couteau
- Lame fixe préférée (robuste)
- Full tang (lame traversante)
- Acier carbone (affûtage facile) ou inox (entretien)
- 10-15 cm de lame

### Cordage
- Paracord 550 : polyvalent, résistant
- Ficelle naturelle : faisable sur place
- Utilisations : abri, pièges, réparations, arcs

## 9. Environnements Spécifiques

### Forêt tempérée
- Ressources abondantes
- Abris faciles
- Eau souvent disponible

### Montagne
- Altitude : froid, hypoxie
- Météo changeante
- Terrain dangereux

### Désert
- Chaleur jour, froid nuit
- Eau critique
- Déplacement nocturne

### Jungle/Tropical
- Eau abondante (purifier !)
- Nourriture disponible
- Humidité, maladie, prédateurs

### Arctique/Neige
- Froid extrême
- Isolation cruciale
- Neige = eau (fondre)

## 10. Psychologie de Survie

### Facteurs de succès
- Volonté de vivre
- Adaptabilité
- Optimisme réaliste
- Compétences pratiques

### Gestion du stress
- Respiration contrôlée
- Tâches immédiates (action)
- Routine quotidienne
- Objectifs petits et atteignables

### Erreurs courantes
- Panique, décisions hâtives
- Surestimation capacités
- Déplacement sans plan
- Négliger l'hydratation
